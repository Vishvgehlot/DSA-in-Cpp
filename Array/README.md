# Array Programs (C++)

This folder contains basic array operations implemented in C++.

---

## Files

- **2-D-Array.cpp** – Demonstrates declaration, initialization, and traversal of a 2-dimensional array.
- **IncreasingSizeOfArray.cpp** – Shows how to increase array size using dynamic memory allocation.
- **Initialization.cpp** – Covers different ways of initializing arrays in C++.
